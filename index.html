!!! DENNE HJEMMESIDE ER AI GENERERET VIA FIGMA MAKE !!!

Denne side demonstrerer, hvordan generativ AI kan bruges til at skabe designkomponenter, som automatisk bliver implementeret i React + TypeScript.
Målet er at undersøge, hvor langt AI kan hjælpe i designprocessen – og hvor menneskelig struktur, vurdering og implementering stadig er nødvendig.
